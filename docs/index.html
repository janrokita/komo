Komo

Komo is a simple and lightweight framework for creating a Spotlight style popup on your website.

Official website: komo.js.org 

function callback(v) {
  console.log(v);
}

Komo(
  {
    darkMode: true,
  },
  callback
);

The default key to open Komo is CMD + K

Read the documentation to learn more!
